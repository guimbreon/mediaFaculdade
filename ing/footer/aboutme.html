College GPA

While I was in college, several colleagues and friends asked me how to calculate the averages for courses and the overall GPA. I usually teach them how to do it, but over time, with many courses taken, it becomes difficult to keep track of everything.

So I decided to create this project to help ANYONE in college easily calculate their GPA!

Project Description

The site allows you to calculate the average of individual assessments or the overall course GPA.

Site Features

Registration and Login

  • Registration and Login Page: Allows users to create an account and log in.
    • Registration: Users can create an account by providing a username and password.
    • Login: Users can log in using their username and password.

Individual Grade Calculator

  • Input of Assessments: Users can enter the number of assessment parts.
  • Dynamic Fields: Dynamic generation of fields for entering assessment values and percentages.
  • Final Grade Calculation: Calculation of the final grade based on the values and percentages entered.

Overall GPA Calculator

  • File Import: Allows the import of Excel files containing course data and respective grades.
  • Manual Input: Allows manual entry of courses, year, semester, ECTS, and grade.
  • GPA Calculation: Calculation of the weighted average of courses based on ECTS.
  • Export of Results: Ability to export results to an Excel file.

File Structure

HTML

  • index.html: Home page of the site.
  • logReg.html: Registration and login page.
  • calcIndiv.html: Page to calculate individual grades.
  • calcTodo.html: Page to calculate overall GPA.

JavaScript

  • index.js: Functions to check login on the home page.
  • logReg.js: Functions for user registration and login.
  • logged.js: Functions to check and manage user login status.
  • calcIndiv.js: Functions to calculate individual grades.
  • calcTodo.js: Functions to calculate the overall GPA of multiple courses.

Instructions for Use

  1. Access the Site: Open the index.html file in your browser or go HERE.
  2. Register or Log In: On first use, you need to create an account on the registration and login page (logReg.html).
  3. Calculate Individual Grades: After logging in, go to the individual grade calculation page (calcIndiv.html) and follow the instructions to enter assessment values.
  4. Calculate Overall GPA: Go to the overall GPA calculation page (calcTodo.html) and choose between importing an Excel file with course data or manually entering course details.
  5. Save Results: Use the export option to save results in Excel format.
